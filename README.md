README - Gestion de Paises 

 

Descripción del programa: 

Este proyecto es una aplicación de consola en Python que permite gestionar información de países utilizando un dataset en formato CSV.  

Incluye funcionalidades como: agregar países, actualizar datos, buscar, filtrar, ordenar y calcular estadísticas. 

 

Instrucciones de uso: 

1. Ejecutar el archivo principal del programa con: python main.py 

2. Utilizar el menú interactivo para seleccionar opciones. 

3. Cargar o guardar datasets en formato CSV desde el menú. 

4. Los datos se almacenan como lista de diccionarios. 

 

Ejemplos de entradas y salidas: 

- Agregar país: 

  Entrada: 

      nombre: Chile 

      población: 19600000 

      superficie: 756102 

      continente: América 

  Salida: 

      "País agregado correctamente." 

 

 

 

- Búsqueda parcial: 

  Entrada: "arg" 

  Salida: 

      Argentina 

- Filtrar por continente: 

  Entrada: America 

  Salida: 

      Argentina, Brasil, Chile 

 

- Estadísticas: 

  Salida: 

      País con mayor población 

      País con menor población 

      Promedio de superficie 

      Cantidad por continente
